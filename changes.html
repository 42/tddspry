Changes in tddspry¶

Release 0.3.1 (latest release)¶

  • Updates documentation (adds Changes in tddspry).
  • Adds tddspry.get_version function to get human-readable version of your tddspry installation.

Release 0.3¶

  • Adds documentation.
  • Re-writes init database connection for :original: and custom databases in tddspry.django.DatabaseTestCase class.
  • Adds object of django.test.Client to tddspry.django.HttpTestCase class.
  • Rewrites test helpers.
  • Adds django-nosetests.py utility.

Release 0.2.3¶

Warning

Do not use this version of tddspry for testing your projects. This version may corrupt data in your database.

  • Fixes for testing with in-memory SQLite3 databases (again).

Release 0.2.2¶

Warning

Do not use this version of tddspry for testing your projects. This version may corrupt data in your database.

  • Fixes for testing with in-memory SQLite3 databases.

Release 0.2.1¶

Warning

Do not use this version of tddspry for testing your projects. This version may corrupt data in your database.

  • First refactoring of tddspry.django.helpers module.

Release 0.2¶

Warning

Do not use this version of tddspry for testing your projects. This version may corrupt data in your database.

  • Full refactoring of tddspry library:
    • Adds tddspry.NoseTestCase class.
    • Rewrites old DbMock and TwillMock classes to DatabaseTestCase and HttpTestCase.
    • Removes tddspry.mock module and adds mock python module to requirements.

Release 0.1¶

Warning

Do not use this version of tddspry for any reasons. This version is deprecated.

  • Initial release.
